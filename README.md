#CronosCube
Hier landen alle Daten um eine Art Cronos Cube zu bauen. 

Liste der Dateien:
- Arduino(ESP8266) für allgemeine Steuerung
- Vektor dateien für das Gehäuse
- Matterial Liste
- Stromlaufpläne

Gehäuse wird 3D Gedruckt
